# مراجعة المنصة وخطة الإكمال

## نتيجة المراجعة
الإصدار السابق كان **هيكلاً أولياً فقط** وليس منصة مكتملة للتشغيل: الواجهة كانت تعرض لوحة واحدة، ولم تكن مسارات التقييم، إدارة البرامج والقوالب، التصدير، أو تخزين التغييرات متاحة للمستخدم في المعاينة. كما أن الخادم يحتاج PostgreSQL/Docker؛ وDocker غير متاح في بيئة المعاينة.

## ما تم توفيره للتشغيل الفوري
تم إنشاء `preview.html` كنسخة مستقلة تعمل بالكامل داخل المتصفح، بلا خادم أو Docker، وتخزن البيانات التجريبية في `localStorage`:

- تسجيل دخول بأدوار: مدير، مدرب، متدرب.
- لوحات تحكم حسب الدور.
- عرض وإضافة وإدارة البرامج، المستخدمين وقوالب التقييم.
- تقييم يومي ونهائي بأسئلة متعددة الأنواع، وحساب تقدم ومتوسط.
- إشعارات: قراءة الكل وحذف الإشعار.
- تقرير برنامج مع تصدير CSV وطباعة.
- جميع التعديلات في نسخة المعاينة محلية وتجريبية.

## قيود مهمة قبل النشر الفعلي
- بيانات المعاينة ليست قاعدة PostgreSQL مشتركة ولا تصلح للإنتاج.
- إرسال بريد/SMS، Refresh Token، رفع ملفات، PDF عربي احترافي، سجل تدقيق، وجدولة التذكيرات تحتاج إكمال الربط بالخادم والخدمات الخارجية.
- يلزم نشر PostgreSQL وBackend في بيئة تستضيف Docker أو قاعدة PostgreSQL مُدارة.

## تشغيل مشروع الإنتاج
اتبع README بعد تثبيت Docker محلياً أو توفير PostgreSQL خارجي. لا تفتح `client/index.html` مباشرة؛ شغّل Vite عبر `npm run dev`.
